# reprorepo: Assessing Reproducibility Checklist

This repository contains issues with checklists to assess the reproducibility of a data science project. The checklists contains questions aiming to identify what can be added to the project and its documentation to make it more reproducible. While the initial goal is for team members to use the checklist to have a discussion on reproducibility, eventually they can convert checklist items into actual issues that can be refined and addressed.
